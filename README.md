# wsCode cli

一个基于 TypeScript 和 Ink 的现代化 CLI 工具，集成了 AI 助手和强大的文件系统工具。

## 功能特性

- 🤖 集成 Anthropic Claude 模型 (通过 OpenRouter)
- 🎨 美观的 Ink React 终端界面
- 📁 强大的文件系统工具（文件读写、目录操作、glob 搜索）
- 🔍 高效的文本搜索和 grep 功能
- 🛠️ 可扩展的工具系统架构
- ✅ 完整的单元测试覆盖
- 🔧 TypeScript + Vitest 开发环境

## 安装和设置

### 从 npm 安装（推荐）

```bash
npm install -g wscode-cli
```

### 打开

```bash
wscode
```
